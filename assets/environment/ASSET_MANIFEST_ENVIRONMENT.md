# Titan Blocks Top-Down Environment Asset Manifest

Drop these into the game project under:

`/assets/environment/`

These are individual PNG files, not a combined sheet.

Recommended use:
- Use the terrain tiles as repeating ground textures.
- Use rock, foliage, wall, crystal, and base-pad PNGs as individual map props.
- Keep spawn zones readable and intentional. Do not hide enemies behind large foreground props.
- Make enemies enter from clear lanes, gates, caves, portals, or map edges.
- Use the spawn gate asset for predictable enemy entry points.
- Use cover wall and boulders for tactical movement, but keep enough open arena space for dodging.
