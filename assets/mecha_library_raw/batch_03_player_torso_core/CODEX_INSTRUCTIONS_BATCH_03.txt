Titan Blocks Premium Mecha Assets, Batch 03
Player torso, chest, core, spine, reactor, and damage wrap assets

Purpose:
This batch contains 10 individual top-down mecha assets for the main player body system. These are intended for the Player Mecha phase, specifically torso, chest, core, spine, reactor, hip connector, and damage variant visuals.

Camera rule:
Every asset in this batch is intended to be 100% top-down / direct overhead, matching the V4 pack camera logic. Do not treat these as side-view or isometric art.

Background rule:
These files use PNG-style checkerboard/transparent presentation from generation. In final game integration, Codex should treat checkerboard pixels as background/alpha cleanup if needed. Do not preserve checkerboard backgrounds in the final game render.

Gameplay rule:
Do not change gameplay mechanics. Preserve movement, hitboxes, arm motion, leg motion, gun rotation, recoil, dash, power-ups, scoring, music, HUD, enemy AI, and boss logic.

Integration rule:
Do not paste these as one full character sheet. These are individual assets. Sort each asset by function into the final player wrap folder.

Player rule:
The original V4 block/square player remains the animation skeleton. These torso/core pieces should wrap over the existing torso/body transforms and support the existing animations.

Recommended final folder:
assets/player_wrap_v6/body_wraps/
